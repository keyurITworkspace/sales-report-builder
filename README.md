# Sales Data Analysis App

This is a simple data analysis project built using Streamlit. The app helps in analyzing sales data and generating useful insights through visualizations.

## Features
- Upload and view sales dataset
- Analyze total revenue and sales trends
- Filter data based on conditions
- Visualize data using charts (matplotlib)
- Identify top-performing products

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## Dataset
The app uses a CSV file containing sales data such as product, price, quantity, and date.

## Purpose
This project is built for analysing sales data and take best decision.
